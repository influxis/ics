Packagecom.adobe.rtc.archive
Classpublic class ArchiveManager
InheritanceArchiveManager flash.events.EventDispatcher
ImplementsISessionSubscriber

The fundamental class behind archive and playback. This can retrived from your connectsession. This class contains APIs for recording and playback. It also throws events on various property changes such as totaltime, currenttime etc.

archiveIDproperty 
archiveID:String  [read-write]

This gives the folder name that contains recording files. If you want to have multiple recordings at the same time without overwrite, you should assign different archiveID

The default value is null.
